### 🔴 BUG-001 — Критический | Неверный тип мероприятия на карточке

**Модуль:** Список мероприятий → Карточка  
**Severity:** Critical  
**Priority:** P1  

**Описание:**  
После создания мероприятия типа «Конференция / митап» тег на карточке отображает **«Хакатон»** вместо «Конференция / митап».

**Шаги воспроизведения:**
1. Нажать «+ Создать мероприятие»
2. Выбрать тип «Конференция / митап»
3. Заполнить название, дату, описание, место
4. Нажать «Создать»
5. Посмотреть тег на карточке в списке

**Ожидаемый результат:** Тег «Конференция / митап»  
**Фактический результат:** Тег «Хакатон»

---

### 🔴 BUG-002 — Критический | Дата не сохраняется при изменении через интерфейс

**Модуль:** Создание мероприятия → Шаг 2 «Основная информация»  
**Severity:** Critical  
**Priority:** P1  

**Описание:**  
Поле «Дата проведения» в визарде создания некорректно обрабатывает ввод с клавиатуры. При попытке изменить дату по умолчанию (текущий день) через ручной ввод изменения не применяются. Дата сохраняется как дата открытия формы (17.07.2026) вместо введённой (25.07.2026).

**Шаги воспроизведения:**
1. Открыть форму создания мероприятия
2. Перейти на шаг 2
3. Попытаться ввести дату «25.07.2026» в поле «Дата проведения» через клавиатуру
4. Нажать «Далее» → проверить preview на шаге 4

**Ожидаемый результат:** Дата «25 июля 2026 г.»  
**Фактический результат:** Дата «17 июля 2026 г.» (дата создания)

**Примечание:** Программный ввод через `form_input` изменяет значение в DOM, но UI не реагирует на изменение — возможно отсутствует обработчик `onChange`.

### 🔴 BUG-003 — Критический | Удаление мероприятия без подтверждения

**Модуль:** Список мероприятий → Кнопка удаления (🗑)  
**Severity:** Critical  
**Priority:** P1  

**Описание:**  
Нажатие кнопки-корзины на карточке мероприятия удаляет его **мгновенно и необратимо** без диалога подтверждения. Отсутствует паттерн «Вы уверены?». Нет undo/восстановления.

**Шаги воспроизведения:**
1. Перейти на страницу списка мероприятий
2. Нажать кнопку 🗑 на любой карточке
3. Наблюдать результат

**Ожидаемый результат:** Модальный диалог «Удалить мероприятие "..."? Это действие необратимо. [Отмена] [Удалить]»  
**Фактический результат:** Мероприятие удаляется мгновенно без предупреждения

### 🟠 BUG-004 — Высокий | Контент вкладок не адаптирован под тип «Конференция/митап»

**Модуль:** Страница администратора → все вкладки  
**Severity:** Major  
**Priority:** P1  

**Описание:**  
На странице управления мероприятием типа «Конференция / митап» все вкладки содержат контент, заточенный под хакатон:
- Вкладка **«Команды»** → «ДОБАВИТЬ КОМАНДУ» (должно быть «ДОБАВИТЬ СПИКЕРА/ДОКЛАД»)
- Вкладка **«Критерии»** → «Для хакатона — критерии оценки команд из положения»
- Вкладка **«Критерии»** → Тип критериев: «Только для команд»
- Вкладка **«Статистика»** → Фильтр «Только команды»

**Ожидаемый результат:** Контент вкладок соответствует типу «Конференция/митап» (спикеры, доклады, секции)  
**Фактический результат:** Всё содержимое дублирует логику хакатона
